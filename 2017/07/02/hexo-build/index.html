使用Hexo + NexT搭建静态博客

发表于 2017-07-02

本站使用 Hexo 来搭建，使用 NexT 主题，这里记录一下搭建本站的整个过程，仅供参考。

1. 环境准备

下载安装 Node.js 和 Git(for windows) ，我的搭建环境是 win10-64bit

1.1. Node.js安装

官网下载地址：https://nodejs.org/en/ 下载 v* LTS Recommended For Most Users

1.1.1 设置环境变量

安装 Node.js,并设置环境变量 Path 的值为相应的安装路径，我是装在了 D 盘的 nodejs 目录下

若生成了 express 和 express-generator 两个文件夹，则表示安装好了

1.1.3 新建项目

若新建成功，则在 nodejs 目录下会生成 newproject 目录，其目录大致包括以下文件

1.1.4 安装模块

进入 newproject 目录下, 安装模块

1.1.5 启动项目

在浏览器访问 http://localhost:3000/ 若测试出现页面 Welcome to Express ，表示 Node.js 安装成功，并可以正常部署项目

可输入以下命令查看版本号

2.6 配置 NexT 主题

3.1 设置语言
