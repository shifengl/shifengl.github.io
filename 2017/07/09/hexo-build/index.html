链接放置在 social 字段下，一行一个链接。其键值格式是 显示文本: 链接地址

设定链接的图标，对应的字段是 social_icons 。其键值格式是 匹配值: 图标名称，匹配键与上一步所配置的链接的显示文本相同（大小写严格匹配）， enable 选项用于控制是否显示图标

设置代码高亮主题

NexT 使用 Tomorrow Theme 作为代码高亮，共有5款主题可选择。 NexT 默认使用白色的 normal 主题，可选的值有 normal，night，night blue，night bright，night eighties 。在主题配置文件中修改

文章添加音乐链接

Hexo 支持解析 markdown 语法，因此每篇博文都是以 .md 结尾的文件。而 markdown 又支持如表格、脚注、内嵌 HTML 等，所以在 .md 文件中直接添加 HTML 代码

网易云音乐、虾米音乐都可以生成内嵌音乐的 HTML 代码，复制粘贴到 .md文件中即可

配置 sshkey

连续按几个回车，密码为空，然后在 C:\Users\Administrator.ssh 下，得到两个文件 id_rsa 和 id_rsa.pub

若出现问题，应该是原本已存在 Github 的sshkey，请自行搜索解决方案清除后重试
